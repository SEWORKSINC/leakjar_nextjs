# Changelog

All notable changes to LeakJar will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Next.js 15.5
- User authentication with Supabase
- Role-based access control (Admin/User)
- Open search functionality for leaked data
- Real-time dashboard with analytics
- Browser statistics visualization
- Country-based analytics with heatmap
- Operating system statistics
- Domain monitoring capabilities
- Email and URL monitoring
- API key management
- Team collaboration features
- Notification system (email/webhook)
- Data export functionality
- Admin panel for user management
- Domain verification system
- ClickHouse integration for analytics
- Responsive UI with Tailwind CSS
- Interactive data grids with TanStack Table
- Chart.js visualizations
- World heatmap for geographic data

### Security
- Implemented secure authentication flow
- Added JWT-based session management
- Implemented role-based authorization
- Added API key authentication
- Secure password hashing with bcryptjs

- Initial release
